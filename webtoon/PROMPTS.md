# 웹툰 학습만화 — 프롬프트 & 수정 매니페스트

이 문서는 「코봇 키우기」 본문에 들어간 웹툰 이미지들을 **나중에 수정·재생성**할 때 쓰는 백업/가이드다.
생성 도구: **M365 Copilot Create — Comic 에이전트**(GPT Image 기반).

---

## 0. 먼저 알아둘 것 (중요)

- **GPT Image는 비결정적이다.** 같은 프롬프트를 다시 넣어도 **똑같은 그림은 안 나온다**(캐릭터·구도가 미묘하게 달라짐). 프롬프트 백업은 "비슷한 새 그림"을 위한 것이지 "원본 재현"용이 아니다.
- **원본을 그대로 두고 부분만 고치려면 → 인라인 편집이 최선.** 원본이 생성된 Comic 대화에서 이미지의 **편집(Image edit)** 버튼을 누르고 "5번 패널 말풍선만 플로봇 것으로 바꿔줘" 처럼 지시하면 화풍·캐릭터를 유지한 채 고칠 수 있다.
- 따라서 진짜 자산은 **① 원본 PNG(이 폴더) + ② 아래 캐릭터 스펙 + ③ Comic 대화**다.

---

## 1. 수정 워크플로 (3가지)

1. **부분 수정(권장)** — 해당 이미지가 만들어진 Comic 대화를 열고 → 이미지 위 **편집** 버튼 → 바꿀 부분만 자연어로 지시. 원본 화풍 유지.
2. **전체 재생성** — 아래 캐릭터 스펙 + 재생성 템플릿(§5)으로 새로 그린다. 화풍이 살짝 달라질 수 있으니 시리즈 일관성이 중요하면 1번을 우선.
3. **텍스트만 수정** — 그림 속 한글 오타가 문제면, GPT Image 재생성보다 **이미지 편집기(예: 그림판/포토샵)로 말풍선 글자만 덮어쓰기**가 빠르고 안전하다. (GPT Image는 한글 오타가 잦음 — 프롤로그는 이 때문에 3회 재생성함.)

### 현재 Comic 대화 URL (표정 시트·아바타가 생성된 세션)
```
https://m365.cloud.microsoft/chat/agent/T_5a4c6de3-3357-18f1-60cf-a6bf958c019d.f5e8a0e5-0c06-4398-96e3-3015799550ec.gpt.020437d6-f3c8-4978-b6dc-456117b92e8e/conversation/772f2241-9815-46c8-8a44-298a0bbc5713
```
> **이 대화(772f2241)에 프롤로그·6개 부 웹툰의 원본 프롬프트가 모두 들어 있다.** 즉 표정 시트·아바타뿐 아니라 7편 전체가 이 한 대화에서 제작됐다. 인라인 편집하려면 이 URL을 열고 해당 이미지의 **편집** 버튼을 쓰면 된다. verbatim 프롬프트는 아래 §6 참조.

## 2. 캐릭터 디자인 스펙 (재생성의 핵심 — 반드시 유지)

**코봇(Cobot)** — 우리가 키우는 AI 신입사원. 판단·추론으로 일을 끝내는 동료.
- 2등신 치비, **흰색 + 파랑** 몸체, 머리에 안테나, 가슴에 **"코봇" 사원증**.
- 표정 풍부, 밝고 친근. 정면 살짝 측면.

**플로봇(Flobot)** — 정해진 절차를 착착 수행하는 로봇(Workflows). 컨베이어벨트 담당.
- 코봇과 같은 화풍, **민트/청록 계열** 몸체, 무한궤도/절차 아이콘 느낌, 가슴에 **"플로봇" 명찰**.
- 체크리스트·순서도를 들고 안내하는 포즈가 어울림.

**공통 화풍:** 파스텔톤, 부드러운 **웹툰 셀(cel) 스타일**, 따뜻한 색감.

## 4. 검증된 verbatim 프롬프트 (이 세션에서 실제 사용)

### 4-1. 코봇 표정 시트 (`cobot-expression-sheet.png`)
```
앞 웹툰들과 동일 화풍·동일 코봇 캐릭터로 '코봇 표정 시트' 한 장을 만들어 주세요.
흰 배경에 3x3 격자, 같은 코봇의 다양한 표정·포즈 9컷: 웃으며 엄지척 / 검지 세워 설명 /
생각하며 턱에 손 / 놀란 표정 / 윙크하며 손 흔들기 / 두 손으로 강조 / 차분한 미소 /
자신감 팔짱 / 응원하듯 주먹 쥐기. 각 컷 상반신만, 말풍선·글자·텍스트·테두리 없이
캐릭터만, 파스텔 셀화풍.
```

### 4-2. 플로봇 표정 시트 (`flobot-expression-sheet.png`)
```
같은 방식으로 '플로봇 표정 시트'를 만들어 주세요. 앞 웹툰들에 나온 플로봇(정해진 절차를
착착 수행하는 로봇 캐릭터)과 동일한 캐릭터·동일 화풍으로, 흰 배경에 3x3 격자 9컷.
다양한 표정·포즈: 웃으며 엄지척 / 검지 세워 설명 / 체크리스트 가리키며 안내 / 놀란 표정 /
윙크하며 손 흔들기 / 두 손으로 강조 / 차분한 미소 / 자신감 팔짱 / 응원하듯 주먹 쥐기.
각 컷 상반신만, 말풍선·글자·텍스트·테두리 없이 캐릭터만, 파스텔 셀화풍. 명찰에는 '플로봇'.
```

### 4-3. 단독 아바타 아이콘(참고용 — 시트 방식으로 대체됨)
```
지금까지 그린 웹툰들과 '똑같은 화풍·똑같은 코봇'으로 캐릭터 아바타 아이콘 한 개.
용도: 책 본문 '코봇 한마디' 말머리 아이콘.
- 코봇 단독. 상반신만. 정면에서 밝게 웃으며 한 손 들어 인사.
- 외형은 웹툰과 동일: 2등신 치비, 흰색+파랑 몸체, 머리 안테나, 가슴에 "코봇" 사원증.

- 정사각형, 캐릭터를 가운데 크게.
- 그림 안에 어떤 텍스트도 넣지 말 것(사원증의 "코봇" 글자만 허용).
```

## 6. 프롤로그 · 6개 부 웹툰의 원본 verbatim 프롬프트

아래는 위 Comic 대화(772f2241)에서 **실제로 보낸 프롬프트 원문**이다. 한 글자도 바꾸지 않고 그대로 옮겼다. 재생성·편집 시 이 원문을 기준으로 삼는다.

